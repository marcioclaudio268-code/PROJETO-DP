# Template Excel V1

Arquivo gerado: `planilha_padrao_folha_v1.xlsx`

## Proposito das abas

- `PARAMETROS`: dados gerais da competencia e do arquivo.
- `FUNCIONARIOS`: cadastro auxiliar para identificar colaboradores e matriculas.
- `LANCAMENTOS_FACEIS`: aba principal de preenchimento humano, sem exigir rubrica tecnica.
- `MOVIMENTOS_CANONICOS`: aba tecnica preparada para a transformacao futura em modelo canonico.
- `PENDENCIAS`: fila de ambiguidades e bloqueios que nao podem seguir silenciosamente.
- `LISTAS`: listas auxiliares para dropdowns e validacoes.

## Regra operacional

- O escritorio deve preencher primeiro `PARAMETROS`, `FUNCIONARIOS` e `LANCAMENTOS_FACEIS`.
- `MOVIMENTOS_CANONICOS` e `PENDENCIAS` existem para a esteira tecnica e para revisao controlada.
- O template nao gera TXT e nao decide rubrica automaticamente.
